# Mercenary Unit Recruitable
Array of uint16_t encoding if a unit is recruitable (1) at the mercenary post or not (0). Only applies to human players.

| Index | Unit              |
| :---: | :---------------- |
| 0     | Arab Archer       |
| 1     | Slave             |
| 2     | Slinger           |
| 3     | Assassin          |
| 4     | Horse Archer      |
| 5     | Arab Swordsman    |
| 6     | Fire Thrower      |
